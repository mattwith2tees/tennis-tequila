## Notes for Event Planning

1. **Weather Contingency:** Indoor capacity should accommodate outdoor VIP ticket holders if needed
2. **Flow:** VIP areas should have dedicated bar service or priority lines
3. **Signage:** Reserve tables with "RESERVED - VIP" signage
4. **Server Ratio:** 1 dedicated server per 2-3 VIP tables recommended
5. **Bottle Service:** Premium option for lounge sections ($150-300 bottles)
